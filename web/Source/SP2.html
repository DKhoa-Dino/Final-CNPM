How to take care

It can be said that Fittonia velvet is a great choice to make a desktop tree because it is derived from a humid swamp, so the tree is cool and resistant to moisture, can live in an air-conditioned environment, lack of sun.

Water

It is a type of water-loving but still has to depend on the conditions to irrigate the tree, for example, when it is hot, dry, it can be watered every day, but if the weather is wet, do not over water.

Light

Can live with only fluorescent light. However, in order for the tree to have a beautiful and clear color, it is still advisable to let the tree sunbathe for 2-3 days. Pay attention to the velvet shade, it is necessary to avoid strong sunlight.
